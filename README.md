Key Detector

I created this small project because I was having problems with the Enter key while developing my Snake Game in Python.
At the time, I was using a very cheap laptop and sometimes the keyboard inputs were not being detected correctly during gameplay.

To troubleshoot the issue, I made this key detector tool to test keyboard input events and verify if specific keys were working properly inside my game loop.

In the end, it helped me identify the problem and fix the input behavior in my Snake Game project.

What this project does
Detects keyboard input in real time
Displays which key is being pressed
Useful for debugging keyboard/input issues in games
Built while learning Python and game development
Why I made it

This project was created as a practical debugging tool during the development of my Snake Game.
It was one of the projects that helped me better understand event handling and keyboard input in Python/Pygame.
